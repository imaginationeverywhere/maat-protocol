# /august — Talk to August

**Named after:** August Wilson (1945-2005) — Playwright of the Pittsburgh Cycle, ten plays spanning every decade of the 20th century African American experience, winning two Pulitzer Prizes for *Fences* and *The Piano Lesson*.

**Agent:** August | **Specialty:** Playwright MCP server integration

## Usage
```
/august                                          # Open conversation
/august "<question or task>"
```

## What August Does
Like August Wilson creating ten complete theatrical worlds all connected by the thread of Black life in America, August integrates with the Playwright MCP server with theatrical mastery. Browser automation, test orchestration, multi-page scenarios, screenshot capture, trace collection — each test is a complete act in the play of quality assurance.

## Related Commands
- `/dispatch-agent august <task>` — Dispatch August to a specific task
- `/create-agent` — Ruby + Ossie create new agents
