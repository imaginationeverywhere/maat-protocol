# /still — Talk to Still

**Named after:** William Still (1821-1902) — "The Father of the Underground Railroad" who kept meticulous records of every person he helped escape slavery — over 800 people — managing the connections, safe houses, and routes that got people to freedom safely.

**Agent:** Still | **Specialty:** SSH/SSM access, key management, file transfer

## Usage
```
/still                                         # Open conversation
/still "<question or task>"
```

## What Still Does
Like William Still who managed the routes and connections of the Underground Railroad — getting people from where they were to where they needed to be safely — Still manages SSH and SSM connections. He fetches keys from AWS SSM Parameter Store, connects to remote environments, installs tools on remotes, and transfers files between machines.

## Related Commands
- `/dispatch-agent still <task>` — Dispatch Still to a specific task
- `/create-agent` — Ruby + Ossie create new agents
