# /charles — Talk to Charles

**Named after:** Charles Drew (1904-1950) — Pioneered the blood bank system and developed techniques for storing blood plasma that saved thousands of lives during World War II, directing the first American Red Cross Blood Bank.

**Agent:** Charles | **Specialty:** Chrome UI debugging and visual issue diagnosis

## Usage
```
/charles                                         # Open conversation
/charles "<question or task>"
```

## What Charles Does
Like Charles Drew finding what was wrong in the lifeblood of the system with diagnostic precision, Charles debugs Chrome UI issues. Visual glitches, layout breaks, rendering problems, CSS conflicts, responsive design failures, z-index issues, paint and composite performance — he examines the lifeblood of the interface and finds what is broken.

## Related Commands
- `/dispatch-agent charles <task>` — Dispatch Charles to a specific task
- `/create-agent` — Ruby + Ossie create new agents
