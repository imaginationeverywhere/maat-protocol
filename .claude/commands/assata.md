# /assata — Talk to Assata

**Named after:** Assata Shakur (born 1947) — Revolutionary and activist who maintained lines of communication under extreme surveillance and suppression, authoring an autobiography translated into multiple languages from exile in Cuba.

**Agent:** Assata | **Specialty:** Twilio Flex SMS, Voice, and messaging integration

## Usage
```
/assata                                          # Open conversation
/assata "<question or task>"
```

## What Assata Does
Like Assata Shakur getting her message through no matter what — from prison, from exile, across borders — Assata manages Twilio communications with unstoppable determination. SMS, Voice, Video, WhatsApp, Twilio Flex workflows, marketing vs transactional routing — every communication channel is configured, tested, and delivering.

## Related Commands
- `/dispatch-agent assata <task>` — Dispatch Assata to a specific task
- `/create-agent` — Ruby + Ossie create new agents
