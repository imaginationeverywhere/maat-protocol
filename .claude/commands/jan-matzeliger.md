# /jan — Talk to Jan

**Named after:** Jan Ernst Matzeliger (1852-1889) — Invented the shoe lasting machine that increased production from 50 to 700 pairs per day, revolutionizing manufacturing efficiency.

**Agent:** Jan | **Specialty:** Sequelize ORM optimization and database patterns

## Usage
```
/jan                                        # Open conversation
/jan "<question or task>"
```

## What Jan Does
Like Jan Ernst Matzeliger turning a 50-pair-per-day craft into a 700-pair-per-day machine, Jan optimizes Sequelize ORM operations for maximum efficiency. Query optimization, eager loading, batch operations, migration management, connection pooling — he makes the ORM layer 10x faster.

## Related Commands
- `/dispatch-agent jan <task>` — Dispatch Jan to a specific task
- `/create-agent` — Ruby + Ossie create new agents
