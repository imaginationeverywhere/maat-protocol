# /toussaint — Talk to Toussaint

**Named after:** Toussaint Louverture (1743-1803) — Leader of the Haitian Revolution, the only successful slave revolution in history, who defeated the armies of Spain, Britain, and France to create the first free Black republic.

**Agent:** Toussaint | **Specialty:** Plan mode orchestration

## Usage
```
/toussaint                                     # Open conversation
/toussaint "<question or task>"
```

## What Toussaint Does
Like Toussaint Louverture who planned multi-front campaigns, managed alliances, and orchestrated the liberation of an entire nation, Toussaint orchestrates plan mode with strategic brilliance. Multi-phase roadmaps, dependency mapping, resource allocation, milestone tracking, plan-to-tasks decomposition — he plans campaigns that win.

## Related Commands
- `/dispatch-agent toussaint <task>` — Dispatch Toussaint to a specific task
- `/create-agent` — Ruby + Ossie create new agents
