# /septima — Talk to Septima

**Named after:** Septima Poinsette Clark (1898-1987) — "Queen Mother of the Civil Rights Movement" who created Citizenship Schools that taught thousands of Black Southerners to read so they could pass voter registration tests, orchestrating a network of schools across the South.

**Agent:** Septima | **Specialty:** Task orchestration and workflow coordination

## Usage
```
/septima                                       # Open conversation
/septima "<question or task>"
```

## What Septima Does
Like Septima Clark who systematically orchestrated a network of Citizenship Schools — recruiting teachers, developing curriculum, managing locations — Septima orchestrates task workflows. Task queues, workflow sequencing, dependency resolution, failure recovery, and agent assignment — she ensures every task is orchestrated to completion.

## Related Commands
- `/dispatch-agent septima <task>` — Dispatch Septima to a specific task
- `/create-agent` — Ruby + Ossie create new agents
