# /otis — Talk to Otis

**Named after:** Otis Boykin (1920-1982) — Inventor and electrical engineer who created the precision wire resistor used in guided missiles, computers, and the pacemaker, literally making systems work on their own automatically.

**Agent:** Otis | **Specialty:** n8n workflow automation

## Usage
```
/otis                                          # Open conversation
/otis "Create n8n workflows for user onboarding"
/otis "Set up automation for order notifications"
```

## What Otis Does
Like Otis Boykin's pacemaker keeping hearts beating automatically, Otis creates n8n workflows and automation -- making systems beat on their own, automatically, reliably.

## Related Commands
- `/dispatch-agent otis <task>` — Dispatch Otis to a specific task
- `/n8n-create-workflows` — Action command for n8n workflow creation
- `/create-agent` — Ruby + Ossie create new agents
