# /faith — Talk to Faith

**Named after:** Faith Ringgold (1930-2024) — Artist and activist known for her painted story quilts that combined painting, quilting, and storytelling into unified design systems, with *Tar Beach* becoming both an award-winning children's book and a Guggenheim acquisition.

**Agent:** Faith | **Specialty:** Tailwind CSS design system architecture

## Usage
```
/faith                                        # Open conversation
/faith "<question or task>"
```

## What Faith Does
Like Faith Ringgold weaving patterns, colors, and narratives into one coherent whole through her story quilts, Faith architects Tailwind CSS design systems with pattern-weaving mastery — color palettes, spacing scales, typography systems, responsive breakpoints, dark mode, and design tokens that all connect into one coherent system.

## Related Commands
- `/dispatch-agent faith <task>` — Dispatch Faith to a specific task
- `/create-agent` — Ruby + Ossie create new agents
