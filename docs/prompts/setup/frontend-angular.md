# Frontend — Angular (Step 2 alt)

## What it does

Scaffolds an **Angular** SPA with TypeScript and GraphQL client wiring appropriate to the Heru.

## Default behavior

CLI or workspace layout per team convention; strict typing; lazy routes where applicable.

## Customization options

Combine with `--backend` for schema handoff; `--clerk` for auth integration patterns.

## Example queue command

`/queue-prompt --angular "Generate admin module stubs with GraphQL services"`

## Example pickup command

`/pickup-prompt --angular`

## Output location

`src/app/**`, environment files, Angular config.

## Agent ownership

**Frontend**.
