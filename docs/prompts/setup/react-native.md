# React Native / Expo (Step 6)

## What it does

Scaffolds **Expo SDK 52 + Expo Router + NativeWind** with Apollo, Clerk Expo, and Redux-Persist per mobile stack standard.

## Default behavior

`--react-native` and `--expo` are aliases; SafeAreaView roots; FlashList; expo-image.

## Customization options

`--push`, `--eas`, `--apple-maps`, `--mapbox` for full mobile ops.

## Example queue command

`/queue-prompt --expo "Auth flow + tab shell with tenant switcher"`

## Example pickup command

`/pickup-prompt --react-native`

## Output location

`mobile/app/**` or project `app/**` per layout.

## Agent ownership

**Mobile** (primary).
