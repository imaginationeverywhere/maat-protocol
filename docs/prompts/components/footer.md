# Footer component

## What it does

Site **footer** with legal links, socials, and optional newsletter slot.

## Default behavior

Links to Privacy + Terms routes; accessible landmark.

## Customization options

Pair with `--privacy-policy` and `--tos` routes; `--design web` for tokens.

## Example queue command

`/queue-prompt --footer "Add sitemap column + locale switcher stub"`

## Example pickup command

`/pickup-prompt --footer`

## Output location

Shared layout footer component.

## Agent ownership

**Frontend**.
