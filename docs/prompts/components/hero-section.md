# Hero section component

## What it does

Delivers a **hero** module: headline, subcopy, primary/secondary CTAs, optional media.

## Default behavior

Mobile-first; reduced-motion variant; tokenized colors.

## Customization options

`--design web`, `--analytics` for CTA events.

## Example queue command

`/queue-prompt --hero-section "Video background with poster + fallback image"`

## Example pickup command

`/pickup-prompt --hero-section`

## Output location

`frontend/components/marketing` or page-local section.

## Agent ownership

**Frontend**.
