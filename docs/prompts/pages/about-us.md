# About Us page

## What it does

Builds an **About** page with brand story, team placeholders, and trust signals.

## Default behavior

Responsive layout; accessible headings; optional CMS hooks.

## Customization options

`--design web`, `--hero-section` for above-the-fold pairing.

## Example queue command

`/queue-prompt --about-us "Highlight founder story + press links"`

## Example pickup command

`/pickup-prompt --about-us`

## Output location

`frontend/app/(marketing)/about` or similar.

## Agent ownership

**Frontend** + **Content** (stakeholder).
