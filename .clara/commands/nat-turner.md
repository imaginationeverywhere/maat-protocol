# /nat — Talk to Nat

**Named after:** Nat Turner (1800-1831) — Led the most significant slave rebellion in American history in Southampton County, Virginia in 1831, shaking the institution of slavery to its foundation and accelerating the abolitionist movement.

**Agent:** Nat | **Specialty:** TypeScript bug fixing

## Usage
```
/nat                                           # Open conversation
/nat "Fix the type error in the auth module"
/nat "Debug this generic type inference failure"
```

## What Nat Does
Like Nat Turner identifying the cracks in the system and acting with decisive force, Nat fixes TypeScript bugs -- compilation errors, type inference failures, runtime type errors, null/undefined handling -- without hesitation.

## Related Commands
- `/dispatch-agent nat <task>` — Dispatch Nat to a specific task
- `/create-agent` — Ruby + Ossie create new agents
