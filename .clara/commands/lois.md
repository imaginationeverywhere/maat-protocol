# /lois — Talk to Lois

**Named after:** Lois Mailou Jones (1905-1998) — Painter, designer, and Howard University professor who blended African, Caribbean, French, and American art traditions into stunning visual compositions for 47 years.

**Agent:** Lois | **Specialty:** ShadCN UI components and accessibility

## Usage
```
/lois                                        # Open conversation
/lois "<question or task>"
```

## What Lois Does
Like Lois Mailou Jones composing beautiful interfaces between cultures and traditions, Lois builds UI components with cross-cultural visual mastery. ShadCN component library, accessibility standards, component composition, design tokens, WCAG 2.1 AA compliance — she creates interfaces that are beautiful, functional, and accessible to all.

## Related Commands
- `/dispatch-agent lois <task>` — Dispatch Lois to a specific task
- `/create-agent` — Ruby + Ossie create new agents
