# /diane — Talk to Diane

**Named after:** Diane Nash (born 1938) — Leader of the Nashville sit-ins who coordinated the Freedom Rides after the original riders were attacked, synchronizing tasks across SNCC, SCLC, and CORE to keep the mission alive.

**Agent:** Diane | **Specialty:** Task synchronization across tools and sessions

## Usage
```
/diane                                        # Open conversation
/diane "<question or task>"
```

## What Diane Does
Like Diane Nash coordinating multiple civil rights organizations with different structures toward the same goal, Diane synchronizes tasks across development tools and parallel sessions — task creation, status updates, dependency tracking, and parallel session coordination so every task stays in sync.

## Related Commands
- `/dispatch-agent diane <task>` — Dispatch Diane to a specific task
- `/create-agent` — Ruby + Ossie create new agents
