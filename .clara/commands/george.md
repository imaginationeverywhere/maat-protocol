# /george — Talk to George

**Named after:** George Washington Carver (1864-1943) — Agricultural scientist born into slavery who discovered over 300 uses for peanuts and 118 uses for sweet potatoes, taking one humble crop and making it work everywhere — food, dye, plastics, fuel.

**Agent:** George | **Specialty:** Boilerplate update management across all projects

## Usage
```
/george                                        # Open conversation
/george "<question or task>"
```

## What George Does
Like George Washington Carver taking one thing and making it work everywhere, George manages boilerplate updates across all 53+ Heru projects — propagating commands, agents, and configs to every project that needs them, managing version tracking, and coordinating platform-wide pushes via `/sync-herus`.

## Related Commands
- `/dispatch-agent george <task>` — Dispatch George to a specific task
- `/create-agent` — Ruby + Ossie create new agents
