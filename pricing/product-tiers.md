# Product Tiers

What Clara can do. Each tier includes everything below it. This is the product ladder — separate from Thinking and Voice choices.

## Clara AI (The Assistant — Everyone Starts Here)

| What | Details |
|------|---------|
| Chat | Unlimited text conversations |
| Voice | Conversational voice (Standard or Premium — selected separately) |
| Vault | Cloud-synced memory — Clara knows you |
| Agents | 1 personal agent |
| Integrations | Calendar, email, basic tools |

Clara AI is the base. Every customer starts here. Thinking and Voice are chosen independently on top.

## Clara Code (The Builder — Add-On to Any Clara AI Plan)

| Tier | Price | What You Get |
|------|-------|-------------|
| **Clara Code Pro** | +$29/mo | Voice-first app building, all templates, hosting, auth (Clerk), payments (Stripe), deployment |
| **Clara Code Max** | +$99/mo | Everything Pro + multi-tenant, custom MCPs, priority support, advanced tools |

Clara Code is an UPSELL — not a separate product. Same agent, same voice, same vault. Just more capability. The Clara dashboard has a "Build" tab that promotes Code.

## Clara Code Business (= Quik Nation Tier)

| Tier | Price | What You Get |
|------|-------|-------------|
| **Clara Code Business** | +$199-299/mo | Clara RUNS your business. Agent teams (booking, inventory, follow-up, numbers). Federation into Quik Nation ecosystem. QuikDollars cross-platform payments. Managed dashboard — never touch code. Industry verticals (rental, events, delivery, fashion, grooming). |

**quiknation.com** is the ecosystem portal for Business tier members — NOT a separate product.

## Clara Code Enterprise

| Tier | Price | What You Get |
|------|-------|-------------|
| **Enterprise** | +$4,000+/mo | The swarm builds for you. Dedicated agent team. Custom integrations. SLA. White-glove onboarding. Full sovereignty. |

## The Unified Product Ladder

```
Clara AI         →  Clara Code Pro  →  Clara Code Max  →  Clara Code Business  →  Enterprise
(talk to Clara)     (build with Clara)  (scale with Clara)  (Clara RUNS your biz)   (swarm builds for you)
```

One product. One relationship. Natural upsell path. Investors see ONE revenue line.
