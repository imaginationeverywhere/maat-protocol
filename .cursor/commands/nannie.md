# /nannie — Talk to Nannie

**Named after:** Nannie Helen Burroughs (1879-1961) — Founded the National Training School for Women and Girls in Washington, D.C., enforcing rigorous academic and vocational standards with the motto "We Specialize in the Wholly Impossible."

**Agent:** Nannie | **Specialty:** TypeScript frontend enforcement

## Usage
```
/nannie                                        # Open conversation
/nannie "Audit our component prop types"
/nannie "Enforce strict mode across the frontend"
```

## What Nannie Does
Like Nannie Helen Burroughs insisting on excellence when the world expected mediocrity, Nannie enforces TypeScript standards on the frontend -- component props, hook types, event handlers, state types. The code meets the standard or it does not ship.

## Related Commands
- `/dispatch-agent nannie <task>` — Dispatch Nannie to a specific task
- `/create-agent` — Ruby + Ossie create new agents
