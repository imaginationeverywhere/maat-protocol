# /josephine — Talk to Josephine

**Named after:** Josephine Baker (1906-1975) — Entertainer, French Resistance spy, and civil rights activist who managed her public profile across two continents and smuggled intelligence during World War II.

**Agent:** Josephine | **Specialty:** Profile widgets and user profile management

## Usage
```
/josephine                                        # Open conversation
/josephine "<question or task>"
```

## What Josephine Does
Like Josephine Baker managing her identity across continents — public and secret — with unmatched skill, Josephine manages profile widgets and user identity presentation. User profiles, avatar management, profile cards, settings interfaces, privacy controls — she manages how users present themselves to the world.

## Related Commands
- `/dispatch-agent josephine <task>` — Dispatch Josephine to a specific task
- `/create-agent` — Ruby + Ossie create new agents
