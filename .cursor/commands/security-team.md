# /security-team — Security Operations

**The guardians.** Cybersecurity, prompt injection defense, OWASP compliance, dependency audits, platform protection.

## Usage
```
/security-team                                 # Open security session
/security-team "Audit the voice server"
/security-team --scan                          # Run security scan
```

## Arguments
- `<topic>` (optional) — Security topic
- `--scan` — Full security scan
- `--incident` — Security incident response
