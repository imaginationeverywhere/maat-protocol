# /creative-team — Creative Session

**The artists.** Design, content, media, branding, documentation. When you need the people who make it look, sound, read, and feel right.

## Usage
```
/creative-team                                 # Open creative session
/creative-team "Design the QuikNation landing page"
/creative-team "We need a brand refresh"
/creative-team --critique                      # Review and critique existing work
```

## Arguments
- `<topic>` (optional) — Creative brief
- `--critique` — Review mode (constructive feedback on existing work)
- `--brand <name>` — Brand-specific session (load brand context)
- `--brainstorm` — Pure ideation, no constraints

2. Jean-Michel and Virgil set direction
