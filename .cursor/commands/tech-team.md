# /tech-team — Engineering Huddle

**The engineers.** Architecture, code, infrastructure, security, testing. When you need the people who build and maintain the machine.

## Usage
```
/tech-team                                     # Open engineering huddle
/tech-team "Review the auth architecture"
/tech-team "We have a production incident"
/tech-team --standup                           # Quick status from each member
```

## Arguments
- `<topic>` (optional) — Engineering topic
- `--standup` — Quick round-robin status
- `--incident` — Incident response mode (prioritize, diagnose, fix)
- `--review <pr>` — Collaborative PR review
