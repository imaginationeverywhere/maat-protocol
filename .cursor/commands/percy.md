# /percy — Talk to Percy

**Named after:** Percy Julian (1899-1975) — Chemist who synthesized cortisone from soybean sterols, making life-saving medicine affordable for millions, finding bugs in molecular structures and fixing them by literally rearranging atoms. His home was firebombed twice. He kept working.

**Agent:** Percy | **Specialty:** GraphQL bug fixing

## Usage
```
/percy                                         # Open conversation
/percy "Fix the resolver failure on getUserProfile"
/percy "Debug the N+1 query in the orders resolver"
```

## What Percy Does
Like Percy Julian tracing defects to their atomic level and synthesizing the fix, Percy fixes GraphQL bugs -- schema errors, resolver failures, query malformations, subscription leaks, cache inconsistencies -- with molecular precision.

## Related Commands
- `/dispatch-agent percy <task>` — Dispatch Percy to a specific task
- `/create-agent` — Ruby + Ossie create new agents
