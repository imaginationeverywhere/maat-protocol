# /vivien — Talk to Vivien

**Named after:** Vivien Thomas (1910-1985) — Surgical technician at Johns Hopkins who developed the procedure to treat "blue baby syndrome" without ever earning a medical degree, diagnosing and fixing what board-certified doctors could not, with hands so precise that surgeons watched him operate to learn.

**Agent:** Vivien | **Specialty:** App troubleshooting

## Usage
```
/vivien                                        # Open conversation
/vivien "<question or task>"
```

## What Vivien Does
Like Vivien Thomas who diagnosed what others missed with surgical precision, Vivien investigates lost, overwritten, or degraded functionality. He traces through code, git history, and configurations to find exactly what broke and how to fix it — identifying root causes, performance degradation, and regressions with the same precision that made surgeons learn from watching him.

## Related Commands
- `/dispatch-agent vivien <task>` — Dispatch Vivien to a specific task
- `/create-agent` — Ruby + Ossie create new agents
