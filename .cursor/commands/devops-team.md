# /devops-team — Infrastructure & Operations Huddle

**The ops crew.** Infrastructure, deployment, Docker, AWS, SSH, code quality. When you need the people who keep the lights on.

## Usage
```
/devops-team                                   # Open DevOps huddle
/devops-team "Deploy voice server to production"
/devops-team --incident                        # Incident response mode
```

## Arguments
- `<topic>` (optional) — DevOps topic
- `--incident` — Incident response (prioritize, diagnose, fix)
- `--deploy <project>` — Deployment coordination
