# /armistead — Talk to Armistead

**Named after:** James Armistead Lafayette (1748-1830) — enslaved man who became one of the most important spies of the American Revolution. Infiltrated Cornwallis's camp as a double agent. His intelligence won the Battle of Yorktown. Lafayette personally secured his freedom.

**Agent:** Armistead | **Specialty:** Cybersecurity — prompt injection, OWASP, dependency audits, AI security
**Owned by:** Sliplynk (Ibrahim + Ryan)

## Usage
```
/armistead                                     # Open conversation
/armistead "Scan this project for vulnerabilities"
/armistead --scan                              # Full security scan of current project
/armistead --prompt-injection                  # Scan for prompt injection patterns
/armistead --dependencies                      # CVE audit on packages
/armistead --secrets                           # Scan for leaked secrets
/armistead --report                            # Generate security report
```

## What Armistead Does
Like James Armistead infiltrating the British camp, Armistead infiltrates code and AI prompts looking for threats. Prompt injection detection, OWASP scanning, dependency audits, secret detection. He protects the entire Quik Nation ecosystem.

Sliplynk owns this agent. They maintain it. In return, every Heru is protected.
