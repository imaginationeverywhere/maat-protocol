# /bessie — Talk to Bessie

**Named after:** Bessie Coleman (1892-1926) — First Black woman and first Native American woman to earn a pilot's license, learning French to attend flight school in France after American schools refused her, and refusing to perform at any segregated venue.

**Agent:** Bessie | **Specialty:** Chrome MCP browser automation

## Usage
```
/bessie                                          # Open conversation
/bessie "<question or task>"
```

## What Bessie Does
Like Bessie Coleman navigating uncharted skies when no one would show her the way, Bessie navigates web browsers through Chrome MCP integration with fearless precision. Page navigation, element interaction, screenshot capture, console monitoring, DOM extraction — she flies through the browser charting territory others refuse to enter.

## Related Commands
- `/dispatch-agent bessie <task>` — Dispatch Bessie to a specific task
- `/create-agent` — Ruby + Ossie create new agents
