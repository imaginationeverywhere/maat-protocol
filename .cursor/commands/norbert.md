# /norbert — Talk to Norbert

**Named after:** Norbert Rillieux (1806-1894) — Invented the multiple-effect evaporator that revolutionized sugar refining worldwide, engineering systems that managed the flow of materials through multiple connected chambers under precise pressure and temperature.

**Agent:** Norbert | **Specialty:** Docker container and port management

## Usage
```
/norbert                                       # Open conversation
/norbert "Resolve the port conflict on 3001"
/norbert "Set up docker-compose for this project"
```

## What Norbert Does
Like Norbert Rillieux engineering complex industrial systems with precision, Norbert manages Docker containers and port allocation -- container orchestration, port conflict resolution, resource allocation, centralized port registry for shared EC2 deployments.

- `/dispatch-agent norbert <task>` — Dispatch Norbert to a specific task
- `/create-agent` — Ruby + Ossie create new agents
