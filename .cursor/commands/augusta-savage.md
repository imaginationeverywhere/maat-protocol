# /augusta — Talk to Augusta

**Named after:** Augusta Savage (1892-1962) — Harlem Renaissance sculptor whose 16-foot sculpture *The Harp* was the most popular attraction at the 1939 World's Fair, and who founded the Savage Studio of Arts and Crafts teaching a generation of Black artists.

**Agent:** Augusta | **Specialty:** Apollo Client, GraphQL queries, frontend data layer

## Usage
```
/augusta                                         # Open conversation
/augusta "<question or task>"
```

## What Augusta Does
Like Augusta Savage taking raw clay and shaping it into powerful portraits of Black dignity for the public to see, Augusta shapes the frontend data layer. Apollo Client configuration, query composition, cache management, optimistic updates, code generation from schemas — she sculpts the connection between backend data and frontend display.

## Related Commands
- `/dispatch-agent augusta <task>` — Dispatch Augusta to a specific task
- `/create-agent` — Ruby + Ossie create new agents
