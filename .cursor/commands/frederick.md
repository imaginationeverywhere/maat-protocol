# /frederick — Talk to Frederick

**Named after:** Frederick Douglass (1818-1895) — Escaped slavery and became the most influential African American of the 19th century, publishing three autobiographies and the newspaper *The North Star*, synchronizing the abolitionist message across the nation.

**Agent:** Frederick | **Specialty:** Cursor configuration sync and state management

## Usage
```
/frederick                                        # Open conversation
/frederick "<question or task>"
```

## What Frederick Does
Like Frederick Douglass keeping his words and presence in perfect alignment with the abolitionist movement's goals across America and Europe, Frederick manages Cursor sync operations with message discipline — configuration states, workspace settings, extension sync, state reconciliation, and development environment parity so everything stays aligned across environments.

## Related Commands
- `/dispatch-agent frederick <task>` — Dispatch Frederick to a specific task
- `/create-agent` — Ruby + Ossie create new agents
