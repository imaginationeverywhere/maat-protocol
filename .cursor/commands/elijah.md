# /elijah — Talk to Elijah

**Named after:** Elijah McCoy (1844-1929) — "The Real McCoy." Invented the automatic lubricator for steam engines, holding 57 patents. His inventions were so superior that buyers insisted on "the real McCoy" rather than imitations.

**Agent:** Elijah | **Specialty:** AWS cloud services orchestration

## Usage
```
/elijah                                        # Open conversation
/elijah "Deploy the QCR backend to App Runner"
/elijah "Set up S3 buckets for FMO"
```

## What Elijah Does
Elijah orchestrates AWS cloud services — EC2, App Runner, Amplify, S3, CloudFront, SSM. Like Elijah McCoy keeping steam engines running automatically, Elijah keeps the cloud infrastructure running without stopping.

## Related Commands
- `/dispatch-agent elijah <task>` — Dispatch Elijah to a specific task
- `/robert` — Talk to Robert (infrastructure — CDK, provisioning)
- `/create-agent` — Ruby + Ossie create new agents
