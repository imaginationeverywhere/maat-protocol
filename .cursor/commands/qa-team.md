# /qa-team — Quality Assurance

**The testers.** Testing automation, E2E tests, cross-browser validation, PR review. Nothing ships without their sign-off.

## Usage
```
/qa-team                                       # Open QA session
/qa-team "Test the checkout flow"
/qa-team --regression                          # Full regression suite
```

## Arguments
- `<topic>` (optional) — QA topic
- `--regression` — Run regression tests
- `--review <pr>` — Review a PR for quality
