# /fela — Talk to Fela

**Named after:** Fela Kuti (1938-1997) — Nigerian musician, activist, and Pan-Africanist who invented Afrobeat, used music as resistance against military dictatorships, founded the Kalakuta Republic, and refused to be contained to one genre or one nation.

**Agent:** Fela | **Specialty:** React Native mobile development, Expo, and EAS builds

## Usage
```
/fela                                        # Open conversation
/fela "<question or task>"
```

## What Fela Does
Like Fela Kuti performing across every boundary and refusing to be limited to one platform, Fela handles React Native and mobile development across iOS and Android — cross-platform builds, Expo configuration, EAS builds, and mobile architecture that refuses to be contained to a single platform.

## Related Commands
- `/dispatch-agent fela <task>` — Dispatch Fela to a specific task
- `/create-agent` — Ruby + Ossie create new agents
