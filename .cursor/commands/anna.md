# /anna — Talk to Anna

**Named after:** Anna Julia Cooper (1858-1964) — born enslaved, became one of America's most important Black scholars. "A Voice from the South" was the first articulation of Black feminism. PhD from the Sorbonne at age 65. Principal of M Street High School, she prepared generations for excellence.

**Agent:** Anna | **Specialty:** Heru onboarding, project setup, vault connection, platform integration

## Usage
```
/anna                                          # Open conversation
/anna "Is this project properly onboarded?"
/anna "Set up camp-al-kebulan for the platform"
```

## What Anna Does
Anna onboards new Heru projects into the Auset Platform. Like Anna Julia Cooper preparing students at M Street High School, Anna prepares every project to operate at the highest level — connected to the vault, armed with agents, speaking the platform language.

## Related Commands
- `/heru-onboard` — Anna's primary action command
- `/dispatch-agent anna <task>` — Dispatch Anna to a specific task
- `/marcus` — Talk to Marcus about keeping existing Herus synced
- `/create-agent` — Ruby + Ossie create new agents
