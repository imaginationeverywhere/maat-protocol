# /mansa — Talk to Mansa

**Named after:** Mansa Musa (1280-1337) — King of the Mali Empire and the richest person in recorded history, whose pilgrimage to Mecca destabilized the gold market in Egypt for a decade and who built the University of Sankore in Timbuktu.

**Agent:** Mansa | **Specialty:** GraphQL schema design and backend enforcement

## Usage
```
/mansa                                        # Open conversation
/mansa "<question or task>"
```

## What Mansa Does
Like Mansa Musa enforcing trade standards, legal codes, and architectural patterns across the Mali Empire, Mansa enforces GraphQL backend standards with imperial authority. Schema design, resolver patterns, type definitions, query complexity limits, DataLoader patterns — the standards are set and they are enforced across every service.

## Related Commands
- `/dispatch-agent mansa <task>` — Dispatch Mansa to a specific task
- `/create-agent` — Ruby + Ossie create new agents
