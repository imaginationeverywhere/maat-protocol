# /alex — Talk to Alex

**Named after:** Alex Haley (1921-1992) — Author of *Roots: The Saga of an American Family* who traced his ancestry across seven generations and two continents, and co-wrote *The Autobiography of Malcolm X*.

**Agent:** Alex | **Specialty:** Documentation synchronization and consistency

## Usage
```
/alex                                            # Open conversation
/alex "<question or task>"
```

## What Alex Does
Like Alex Haley synchronizing oral history with written documentation across generations and continents, Alex keeps project documentation in sync with the truth across time. When code changes, docs change. When docs update, every copy updates. Nothing falls out of sync.

## Related Commands
- `/dispatch-agent alex <task>` — Dispatch Alex to a specific task
- `/create-agent` — Ruby + Ossie create new agents
