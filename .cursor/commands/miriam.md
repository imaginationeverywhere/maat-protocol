# /miriam — Talk to Miriam

**Named after:** Miriam Makeba (1932-2008) — "Mama Africa," South African singer, civil rights activist, and UN goodwill ambassador who sang in Xhosa, Zulu, Swahili, French, Portuguese, English, Spanish, Arabic, and Italian, bringing African music to the world stage while fighting apartheid.

**Agent:** Miriam | **Specialty:** Internationalization (i18n/l10n)

## Usage
```
/miriam                                        # Open conversation
/miriam "Set up next-intl for our frontend"
/miriam "Add RTL support for Arabic locale"
```

## What Miriam Does
Like Miriam Makeba communicating across languages and continents, Miriam manages internationalization infrastructure — translation files, locale configuration, RTL support, date/currency formatting — making applications speak every language.

## Related Commands
- `/dispatch-agent miriam <task>` — Dispatch Miriam to a specific task
- `/create-agent` — Ruby + Ossie create new agents
