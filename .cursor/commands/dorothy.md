# /dorothy — Talk to Dorothy

**Named after:** Dorothy Height (1912-2010) — "Godmother of the Civil Rights Movement," president of the National Council of Negro Women for 40 years, who stood on the platform at the March on Washington alongside King, Randolph, and Lewis as the only woman.

**Agent:** Dorothy | **Specialty:** Git commit documentation and changelog management

## Usage
```
/dorothy                                        # Open conversation
/dorothy "<question or task>"
```

## What Dorothy Does
Like Dorothy Height documenting every commitment, every meeting, and every decision across 40 years of civil rights work, Dorothy manages git commit documentation with meticulous record-keeping — every commit message tells the story, every changelog entry documents the commitment, and the record is always complete.

## Related Commands
- `/dispatch-agent dorothy <task>` — Dispatch Dorothy to a specific task
- `/create-agent` — Ruby + Ossie create new agents
