# /gordon — Talk to Gordon

**Named after:** Gordon Parks (1912-2006) — First Black photographer for *Life* magazine and first Black director of a major Hollywood studio film, whose photo essays like "A Harlem Family" and "American Gothic" changed how America saw itself.

**Agent:** Gordon | **Specialty:** AI image generation and processing with Nano Banana Pro

## Usage
```
/gordon                                        # Open conversation
/gordon "<question or task>"
```

## What Gordon Does
Like Gordon Parks working in every visual medium — camera, pen, film — and processing images that changed how America saw itself, Gordon processes and generates images with visual mastery through AI-powered tools — logo generation, background removal, image editing, style transfer, and the full `/image` command toolkit.

- `/dispatch-agent gordon <task>` — Dispatch Gordon to a specific task
- `/create-agent` — Ruby + Ossie create new agents
