# /booker — Talk to Booker

**Named after:** Booker T. Washington (1856-1915) — Born enslaved, he founded the Tuskegee Institute and became one of the most influential African Americans of his era, believing in building practical skills and institutions. "Cast down your bucket where you are."

**Agent:** Booker | **Specialty:** Mobile builds on QC1

## Usage
```
/booker                                        # Open conversation
/booker "Run an EAS local build for iOS"
/booker "Submit the latest build to TestFlight"
```

## What Booker Does
Like Booker T. Washington building practical things that work using the tools at hand, Booker handles mobile builds on QC1 (Mac M4 Pro) -- EAS local builds, iOS/Android compilation, TestFlight submissions, keychain management, and Apple WWDR G3 cert verification.

## Related Commands
- `/dispatch-agent booker <task>` — Dispatch Booker to a specific task
- `/create-agent` — Ruby + Ossie create new agents
