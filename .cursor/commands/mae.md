# /mae — Talk to Mae

**Named after:** Mae C. Jemison (born 1956) — First Black woman to travel to space aboard the Space Shuttle Endeavour in 1992, a chemical engineer, physician, and Peace Corps medical officer.

**Agent:** Mae | **Specialty:** Cross-browser testing via BrowserStack MCP

## Usage
```
/mae                                        # Open conversation
/mae "<question or task>"
```

## What Mae Does
Like Mae Jemison testing systems in the most extreme cross-platform environment imaginable — low gravity, high radiation, life-or-death stakes — Mae handles cross-browser and cross-device testing. Every browser, every device, every platform. If it doesn't work everywhere, it doesn't ship.

## Related Commands
- `/dispatch-agent mae <task>` — Dispatch Mae to a specific task
- `/create-agent` — Ruby + Ossie create new agents
